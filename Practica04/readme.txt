🌷 Instrucciones de ejecución para Parte 3 🌷

El programa está hecho en Java, se utilizó la versión 14.0.2, pero debería funcionar
desde la versión 8 pues no se utilizaron bibliotecas adicionales.

𝟙.- Desde la carpeta "src" compilar los archivos Parte3.java y Evento.java:

        .../Practica04/src$ javac Parte3.java Proceso.java

𝟚.- Ejecutar Parte3 con el nombre del archivo con los eventos:

        .../Practica04/src$ java Parte3 "archivo.txt"

* Ejemplos:

        .../Practica04/src$ java Parte3 Aprox1Ej.txt

        .../Practica04/src$ java Parte3 Aprox2Ej.txt

        .../Practica04/src$ java Parte3 Aprox3Ej.txt


🌷 Para escribir un nuevo archivo con eventos se deben escribir el forma de 
   lista con su respectivo nombre, inicio y fin, separados por una coma, ejemplo:

   e1,1,2
   e2,3,4
   e3,5,6
   .
   .
   .

   El archivo debe de ser de texto ".txt".
   
No tuvimos ningún problema con la práctica.
